# Moodboard.exe

Moodboard.exe is an interactive, JavaScript-powered moodboard generator that allows users to choose a mood such as Soft Girl, Y2K, or Dark Academia.

Each mood dynamically generates images, background music, quotes, and customizable color themes to match the selected vibe.

## Features
- Mood-based image generation
- Background music unique to each mood
- Dynamic aesthetic quotes
- Customizable color themes
- Interactive and responsive UI

## Built With
- HTML
- CSS
- JavaScript
